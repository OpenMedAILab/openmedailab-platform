# CHANGELOG

## 0.3.0 - 2026-06-09

### Added

- 新增用户空间，集中展示总览、收藏、申请、任务、贡献、积分流水和个人资料。
- 新增管理员空间总览、协作管理、任务管理、贡献审核、积分流水和审计日志。
- 新增课题卡状态悬浮窗，登录用户可查看已确认参与者 UID。
- 新增任务分配、用户提交贡献、管理员审核贡献并发放任务积分的协作闭环 API。
- 新增用户撤回参与/认领/资助意向 API。
- 新增用户空间与管理员空间协作生命周期执行文档和严格验收标准。

### Changed

- 扩展 `/api/me/dashboard/`，返回任务、贡献和积分流水，作为用户空间统一数据入口。
- 顶部导航登录后将“我的协作”和“我的收藏”合并为“我的空间”，降低拥挤感。
- 管理员课题列表增加“建任务”入口，便于从课题直接进入任务拆分。
- 协作、任务、贡献、积分和审计中的人员展示统一收敛为 UID，不对外暴露用户名或邮箱。
- 收缩前端 API 包装层，只保留当前用户空间和管理员空间实际调用的生命周期方法。

### Fixed

- 管理员审核、任务状态、用户撤回和贡献审核等关键动作统一写入审计日志。
- 任务奖励发放增加幂等保护，避免同一任务重复审核导致积分重复入账。
- 新增空间列表、管理员表格和课题状态悬浮窗的小屏自适应样式。
- 修复管理员功能 tabs 在中等宽度下按钮超出可视范围的问题。

## 0.2.0 - 2026-06-09

### Added

- 课题库列表改为一行一个，并展示关键字段。
- 增加主题筛选、我的收藏页面、个人信息小窗和收藏交互反馈。
- 增加系统版本号与更新日志小弹窗。

### Changed

- 首页课题库引导区降低视觉权重。
- 课题详情文件展示改为优先展示所属主题文件域。
- 文件空间页改为紧凑工作台布局，降低空白感并强化主题概览、文件域策略、空状态和关联课题展示。
- 优化顶部导航、课题详情弹窗和管理表格在不同分辨率下的自适应布局。

### Fixed

- 课题详情和预览中收藏按钮缺少状态反馈的问题。
- 修复课题预览弹窗内收藏反馈显示到模糊遮罩背景中的问题。
- 修复课题预览弹窗内“查看主题文件空间”跳转后弹窗未关闭的问题。

## 0.1.0 - 2026-06-08

### Added

- 完成用户注册、登录、系统统一默认密码恢复和管理员用户管理基础能力。
- 完成课题库、主题文件域、课题导入和协作交互基础 API。
