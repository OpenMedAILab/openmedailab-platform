# CHANGELOG

## 0.4.0 - 2026-06-09

### Added

- 新增管理员文件空间管理器，支持受限服务器目录、目录浏览、新建、编辑、删除、文件上传和目录上传。
- 新增课题生命周期重建设计与执行文档，固定管理员创建、系统内编辑和 Markdown 模板导入流程。
- 新增 Markdown v1 模板、模板复制、文件/文件夹导入预览、关键字段修复和重复识别能力。
- 新增管理员课题列表的阶段、公开状态、课题 ID、来源路径和内容哈希筛选能力。
- 新增用户侧“我的任务”聚合视图，按课题展示申请状态、获批关系和任务结果提交入口。
- 新增管理员“任务审批 / 任务管理”工作流，复用现有申请、课题和任务结果接口完成审批、查看、审核和结题归档。
- 新增任务结果类型，区分阶段性成果与最终结果，为最终结果通过后结题归档提供依据。

### Changed

- 首页口号改为极简文案，并移除多余说明文本。
- 课题详情改为只展示结构化课题信息与主题文件空间，不再展示课题原始文档模块。
- 前端统一使用“文件空间”命名，并简化文件空间页面说明。
- 课题创建入口统一收敛到管理员课题管理；系统内创建默认先落草稿且不公开，发布通过更新同一课题完成。
- 管理员课题表单将数据需求、评价指标、预期成果和文件列表改为结构化输入，减少手写 JSON。
- Markdown 导入默认写入草稿且不公开，新增主题时需要确认并可修改主题名称、slug 和说明。
- 公开课题详情与管理员详情分离，用户侧不再暴露内部来源 payload、内容哈希或本机/内部导入路径。
- 课题阶段收敛为草稿、开放招募、组队中、进行中、暂停和归档，并在管理员课题列表中支持直接下拉调整。
- 用户空间总览卡片与管理员空间保持一致，支持点击进入对应子页面。
- 管理员用户列表改为平台管理员固定第一行，其他用户按 UID 稳定排序。
- 课题状态悬浮卡按收藏、参与、认领和资助分组展示 UID，并保持分组和 UID 排序稳定。

### Fixed

- 修复弹窗打开后底层页面仍可滚动的问题。
- 移除网页端 JSON 导入产品入口和前端调用，API 文档不再暴露旧 JSON 导入路由。
- 修复管理员创建课题可直接通过 POST 公开发布的生命周期绕过问题。
- 修复评分维度 JSON 非法时被静默吞掉的问题，改为明确错误提示。
- 修复未获批用户可绕过前端直接提交任务结果的风险。
- 修复新增课题发布警告、任务管理详情和任务结果提交弹窗的层级遮挡问题。
- 修复任务详情阶段更新会把非归档课题误设为公开的风险。

## 0.3.0 - 2026-06-09

### Added

- 新增用户空间，集中展示总览、收藏、申请、任务、贡献、积分流水和个人资料。
- 新增管理员空间总览、协作管理、任务管理、贡献审核、积分流水和审计日志。
- 新增课题卡状态悬浮窗，登录用户可查看已确认参与者 UID。
- 新增任务分配、用户提交贡献、管理员审核贡献并发放任务积分的协作闭环 API。
- 新增用户撤回参与/认领/资助意向 API。
- 新增用户空间与管理员空间协作生命周期执行文档和严格验收标准。

### Changed

- 扩展 `/api/me/dashboard/`，返回任务、贡献和积分流水，作为用户空间统一数据入口。
- 顶部导航登录后将“我的协作”和“我的收藏”合并为“我的空间”，降低拥挤感。
- 管理员课题列表增加“建任务”入口，便于从课题直接进入任务拆分。
- 协作、任务、贡献、积分和审计中的人员展示统一收敛为 UID，不对外暴露用户名或邮箱。
- 收缩前端 API 包装层，只保留当前用户空间和管理员空间实际调用的生命周期方法。

### Fixed

- 管理员审核、任务状态、用户撤回和贡献审核等关键动作统一写入审计日志。
- 任务奖励发放增加幂等保护，避免同一任务重复审核导致积分重复入账。
- 新增空间列表、管理员表格和课题状态悬浮窗的小屏自适应样式。
- 修复管理员功能 tabs 在中等宽度下按钮超出可视范围的问题。

## 0.2.0 - 2026-06-09

### Added

- 课题库列表改为一行一个，并展示关键字段。
- 增加主题筛选、我的收藏页面、个人信息小窗和收藏交互反馈。
- 增加系统版本号与更新日志小弹窗。

### Changed

- 首页课题库引导区降低视觉权重。
- 课题详情文件展示改为优先展示所属主题文件域。
- 文件空间页改为紧凑工作台布局，降低空白感并强化主题概览、文件域策略、空状态和关联课题展示。
- 优化顶部导航、课题详情弹窗和管理表格在不同分辨率下的自适应布局。

### Fixed

- 课题详情和预览中收藏按钮缺少状态反馈的问题。
- 修复课题预览弹窗内收藏反馈显示到模糊遮罩背景中的问题。
- 修复课题预览弹窗内“查看主题文件空间”跳转后弹窗未关闭的问题。
